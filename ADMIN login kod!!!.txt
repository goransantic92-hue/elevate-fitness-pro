Korak 3.3 — SQL: postavi ulogu admin
SQL Editor → New query.
Ubaci (zameni email svojim):
update public.profiles
set role = 'admin'
where email = 'tvoj-email@example.com';
Run.
Trebalo bi da piše npr. 1 row updated (ako je email tačan).
